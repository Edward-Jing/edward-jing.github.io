# Jingzhi Sun Personal Website

Static personal website for Jingzhi Sun, published through GitHub Pages at:

https://edward-jing.github.io/

## Structure

- `index.html` - homepage and introduction
- `research.html` - advisors and research topics
- `map.html` - visitor map
- `life.html` - hobbies and travel
- `gallery.html` - model and miniature gallery
- `dota2.html` - Steam linked Dota2 profile
- `assets/` - site CSS, JavaScript, and local data fallback
- `images/` - avatar, favicon, and gallery images

The site is intentionally static and does not depend on the old Jekyll academic template.
